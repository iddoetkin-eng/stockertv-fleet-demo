The things that make the whole thing.

No dependencies. No analytics. No build. Everything you just saw is one download — HTML, CSS, JavaScript, a few hand-drawn SVG glyphs, audio synthesized at runtime. Built by a newsroom that cares whether the em-dash has the right tracking.

Typography

Fraunces
Variable serif by Undercase Type. opsz axis 9–144, wght 400–700, roman and italic. Carries every editorial voice on this site — splash, mastheads, compliance hero, pull-quotes, drill modal drop-caps, watermarks.
Inter
Humanist sans-serif by Rasmus Andersson. All UI chrome, body copy, stage subtitles.
JetBrains Mono
Monospace by JetBrains. Every number on the site is tabular-nums so digits don't jitter on live update.

Audio

Every sound, synthesized
The entire audio layer is procedural Web Audio — sine, square, triangle, and noise oscillators through ADSR envelopes and biquad filters. Nothing is sampled. The StockerTV three-note ident, the five exchange-specific bells (NYSE brass, LSE Big-Ben, HKEX gong, TSE cathedral, NASDAQ brass), the gallery bell on compliance pass, the opt-in room tone — every sound is generated at runtime in soundfx.js.

Motion

All animation, guarded
Every procedural animation routes through shouldAnimate(). Under prefers-reduced-motion: reduce, the 4.1s cold open collapses to 900ms, the Market Open Ceremony shrinks to a fade, the curator idle mode stays dormant. The site holds its promise to be usable, not just impressive.
Curator mode
After 45 seconds of no activity, the site begins to host itself — editorial pull-quotes rotate in Fraunces italic, the world map drifts in saturation, the metrics recede. One mouse move and everything returns. The site behaves differently when nobody is watching.

Data & telemetry

Exchange schedules
Bundled as a static file: exchange-schedule.js. Five majors with 2026 holidays and early-close windows. IANA timezone names + Intl.DateTimeFormat handle DST without a library.
Live data
Showcase data is a snapshot from the upstream orchestrator, regenerated on demand. No external API call at page load. No telemetry sent to us. No cookies beyond your theme and sound preferences.

Dependencies

Zero, on purpose
There is no package.json. No node_modules. No build step. No CDN scripts beyond Google Fonts. Every line of JavaScript is one we wrote. Every SVG path is one we drew. The total page weight stays under 500KB per page with room to spare.

Deployment

GitHub Pages
Push to main. Pages rebuilds in ~30s. There is no server. There is no database. A static folder, replayed.
